* **APPLICATION**

  * **Save File**
    * **Save**: CTRL+S
    * **Save As**: SHIFT+CTRL+S
    * **Save Incremental**: CTRL+ALT+S

* **GENERAL**

  * **Delete**
    * Select + X
    * Delete key
    * Right-click context menu
  * **Select**
    * **Select All**: A
    * **Cursor drag selection mode**: W
    * There are different modes you can select from, see left-side icon.
    * **Select an object behind another**: ALT + mouse left-click
  * **Add Object**
    * Shift + A
    * Menu bar: Add
      * Lower-right menu: edit after creating, edit parameters
  * **Object Mode / Edit Mode**
    * Tab
    * Menu bar: left-most dropdown
    * Mode
      * Object Mode: add new objects
      * Edit Mode: edit mesh, add new polygons to current object, etc.
  * **View**
    * ~ Tilda key then select or use shortcut letter (underscored letter in radial menu)
    * Use axes view in upper-right of 3D view (window).
    * Numberpad
    * Alt + Middle mouse button drag
    * Menu: frame selected / frame all
    * **Toggle Orthographic / Perspective**: Numpad 5
    * **Face Orientation**: viewport Show Overlays menu checkbox (upper-right, 2 overlapping circles)
    * **View Statistics (view polycount)**: Viewport > Statistics (enable), will show in upper-left of window.
  * **Cursor**
    * Left-hand menu
    * Left-click to place in scene
    * Shift-S: snap settings radial menu
  * **Move / Translate**: G
    * Press X, Y, or Z after to constrain to that axis
  * **Rotate**: R
    * Press X, Y, or Z after to constrain to that axis
  * **Scale**: S 
    * Press X, Y, or Z after to constrain to that axis
  * **Transform**: upper right, "item" sidebar
  * **Camera Controls**
    * **Rotate**: mouse middle-click
    * **Translate**: Shift mouse middle click
    * **Zoom**: mouse wheel
    * **Focus**: home key
  * **Edit Mode**: SHIFT+Space
    * **Select Vertex**: press 1 (number)
    * **Select Edge**: press 2 (number)
    * **Select Face**: press 3 (number)
    * **Select Tool**: mouse-left click which in window
      * **Add to Selection**: hold SHIFT
      * **Remove from Selection**: hold CTRL
      * **Select All**: A
      * **Select Loop**: ALT+left mouse click
      * **Select Linked**: L (while mouse over)

* **MODELLING**

  * **Select**:

    * **Select All**: A
    * **Select Loop**: ALT+left mouse click
    * **Select Linked**: L (while mouse over)

  * **Extrude**: E 

    * **Extrude Out**: E followed by S (scale)

    * **Inset**: i
      * **Inset Individually** (eg. faces independently): press i twice
      * **Inset again boundary**: B

  * **Bevel**: CTRL+B

  * **Loop Cut**: CTRL+R

  * **Fill**: F
    * **Bridge Edge Loops**: F 
    * **Fill Between Edges**: F
  * **Knife**: K
    * **Angle Constraint**: A
    * **Axis Constraint**: X, Y, Z

  * **Spin**: SHIFT+0

  * **Edge Slice**: CTRL+3

  * **Rip Region**: V

  * **Split**: Y (detaches vertices from neighbours)

  * **Mirror Editing**: Mirror modifier
    * Apply can only be done in Object Mode

  * **Merge**: M (menu opens)

    * **Merge Vertices**: M, select merge option

    * **Merge Faces**: right-click, "Dissolve Faces" at bottom of menu (in edit mode, face view [3]) 

  * ... intersect CTRL F face menu
  * **Duplicate**: SHIFT+D
  * **Recalculate Normal**: SHIFT+N
  * **Open Normals Menu**:  ALT+N
    * **Flip**: Invert (eg. invert faces)
    * **Backface Culling**: per material under Settings > Surface

* **UV MAPPING**

  * **Select**:
    * **Select All**: A
    * **Select Linked**: L
  * **Move Selected UVs to Cursor**:  SHIFT+S, then "Selected to Cursor (Offset) 7"
  * **Projections** (while selected)
    * Unwrap from context menu
    * In "UV" in modelling edit mode
  * **Split** (selection) Y

* **ANIMATION**

  * **Keyframe Property**: K
  * **Insert Frame**: i (after channels are watched)
  * **Delete unused actions / animations**: File > Clean Up > Purge Unused Data
  * **Graph Editor**:
    * **Frame All**:
      * Home
